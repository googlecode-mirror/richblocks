Introdução

Para o desenvolvimento de alguma aplicação utizando o RichBlocks como interface, é necessário fazer o mapeamento dos menus e janelas que esta aplicação irá possuir.

Exemplo:

Se uma determinada aplicação vai possuir o requisito de incluir um usuário no sistema, é interessante que esta opção esteja diponível em algum menu, e que quando esta opção seja selecionada, a janela com o conteúdo de inclusão de usuário seja exibida.

Ainda é possível exibir um menu de botão direito para fazer chamada a uma janela.


Para isso funcionar é necessário editar os arquivos XML de menu e de janelas.


Exemplo de menu:


<?xml version="1.0" encoding="ISO-8859-1" ?>

<MenuInterfaceRia>

               

               <!--RichTools-->

               <menu name="menu_1" value="RichTools" img="img.png">

                       <option name="option1_menu_1" value="Google" windowTarget="google" img="google.gif" shortcut="Alt+G"></option>

                       <option name="option2_menu_1" value="Plano de Fundo" windowTarget="change_background" img="background_image.png" shortcut="Alt+PI"></option>

                       <option name="option3_menu_1" value="Ajuda" windowTarget="help" img="help.png" shortcut="Alt+H" separator="true"></option>

                       <option name="option4_menu_1" value="Sobre..." windowTarget="sobre" img="rich_blocks_ico.png" shortcut="Alt+S"></option>

               </menu>

               

</MenuInterfaceRia>
